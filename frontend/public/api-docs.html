üßÆ Calculate Tariff

POST /tariff/calculate

Calculate import duties and taxes

http://localhost:8000/api/v1/tariff/calculate?hs_code={hs_code}&country={country}&value={value}
Parameter Type Required Description
hs_code string Yes Full HS code (e.g., "8703230010")
country string Yes Destination: CN, EU
value number Yes CIF value in USD

Example Request:

POST /api/v1/tariff/calculate?hs_code=8703230010&country=CN&value=50000

üí± Calculate with Currency Conversion

POST /tariff/calculate-with-currency

Calculate tariff and convert to local currency

http://localhost:8000/api/v1/tariff/calculate-with-currency?hs_code={hs_code}&country={country}&value={value}&from_currency={from}&to_currency={to}
Parameter Type Required Description
hs_code string Yes Full HS code
country string Yes Destination: CN, EU
value number Yes CIF value
from_currency string No Default: USD
to_currency string No Default: USD (CNY, EUR, JPY, etc.)

Example Request:

POST /api/v1/tariff/calculate-with-currency?hs_code=8703230010&country=CN&value=50000&from_currency=USD&to_currency=CNY
